# Ran's Portfolio

福岡を拠点に活動するWebコーダー Ran のポートフォリオサイトです。
静的HTML/CSS/JavaScriptを用いて制作しており、PC・スマホで閲覧可能です。

---

## サイト内容
- **Serviceページ**
提供サービスの紹介
- ホームページ制作（目的に合わせた柔軟な提案）
- ホームページ修正（文言変更や表示崩れ対応）
- ホームページ相談（制作全般の相談）

- **Worksページ**
制作実績の紹介
- 自身のポートフォリオサイト
- スイーツショップ公式サイト（サンプル）

- **Aboutページ**
自己紹介、経歴、趣味など

- **Work Flowページ**
制作の流れ（相談〜コーディング〜確認〜納品）

- **Messageページ**
コーディングの方針や対応力の説明

- **Contactページ**
問い合わせフォーム・SNSリンク

---

## 工夫した点
- **レスポンシブデザイン**
PC・スマホ・タブレットでも見やすいUIを実装
- **サービス・実績の整理**
ページごとに目的別に情報を整理し、直感的に閲覧可能
- **制作フローの可視化**
Work Flowページで作業の流れを分かりやすく提示
- **ユーザー目線の導線設計**
問い合わせや相談ページへの誘導を自然に配置
- **ビジュアルの統一感**
色・フォント・レイアウトを統一してブランド感を演出
- **SNS連携**
Twitterリンクなどで外部情報への接続を簡単に

---

## 技術スタック
- HTML5 / CSS3 / JavaScript
- レスポンシブデザイン（メディアクエリ）
- 静的サイト構築・ポートフォリオ用ページ設計
